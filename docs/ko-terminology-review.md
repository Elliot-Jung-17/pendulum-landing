# 한국어 물리 용어 감수 체크리스트

`index.html`이 영문 원본이며 `assets/i18n-core.js`의 사전으로 `ko.html`을
생성합니다. 용어를 바꾼 뒤에는 `npm run build:ko`를 실행하고 교과서 문맥,
단위, 기호를 함께 확인합니다.

## 기준

- 고등학교 물리학Ⅰ·Ⅱ 및 대학 일반물리에서 널리 쓰는 표기를 우선합니다.
- 기호(`γ`, `λ`, `A_PD`, `ΔE/E₀`)는 번역하지 않고 첫 등장에 뜻을 풉니다.
- 영문 병기가 이해를 돕는 전문 용어는 첫 등장에만 괄호로 덧붙입니다.
- 검토자는 아래 표에 날짜와 근거를 남기고, 미검토 항목을 임의로
  “감수 완료”로 표시하지 않습니다.

## 핵심 용어

| English | 승인 표기 | 상태 | 검토 근거/메모 |
| --- | --- | --- | --- |
| damping | 감쇠 | 감수 대기 | `damping γ`는 `감쇠 계수 γ`로 풀이 |
| driven pendulum | 구동 진자 | 감수 대기 | 외력에 의한 주기 구동 문맥 확인 |
| bifurcation | 분기 | 감수 대기 | 첫 등장에 `분기(bifurcation)` 병기 |
| period doubling | 주기 배가 | 감수 대기 | `period-doubling onset` → `주기 배가 시작점` |
| phase portrait | 위상 초상 | 감수 대기 | 교재에 따라 `위상 공간 궤적` 보조 표기 |
| Poincaré section | 푸앵카레 단면 | 감수 대기 | 고유명사 악센트/한글 표기 통일 |
| Lyapunov exponent | 리아푸노프 지수 | 감수 대기 | `λ`와 함께 표시 |
| energy drift | 에너지 표류 | 감수 대기 | 수치 오차이며 물리적 감쇠와 구분 |
| residual | 잔차 | 감수 대기 | 비선형 반복 수렴 문맥 확인 |
| symplectic integrator | 심플렉틱 적분기 | 감수 대기 | 필요 시 `정준 구조 보존` 설명 |
