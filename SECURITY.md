# Security Policy

Security fixes are applied to the deployed default branch. Historical static
copies may not receive backports.

Please use GitHub's **Security > Report a vulnerability** private advisory for
this repository. If that channel is unavailable, email
`jung.elliot@gmail.com` with the subject `Pendulum landing security report`.
Include the affected commit or URL, reproduction steps, impact, and suggested
mitigation. Do not open a public issue or disclose exploit details before a fix
is available. An initial acknowledgement is targeted within seven days.
